Troubleshooting:

If you see this page but get 404 for other pages:

  • Run: docker-compose down
  • Run: docker-compose up --build -d
  • Check logs: docker-compose logs frontend

File Check:

Run this command to see what files nginx is serving:

docker exec ledger_frontend ls -la /usr/share/nginx/html/
